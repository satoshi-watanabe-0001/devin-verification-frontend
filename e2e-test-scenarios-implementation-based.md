## 注意事項

### テスト実行時の注意点
1. **バックエンドAPI**: 一部のテストはバックエンドAPIが稼働していることを前提としています。APIが利用できない場合は、モックデータを使用するか、エラーハンドリングの確認を行ってください。

2. **ブラウザキャッシュ**: テスト前にブラウザのキャッシュをクリアすることを推奨します。

3. **LocalStorage**: 認証状態のテストではLocalStorageを使用します。テスト前後でLocalStorageをクリアしてください。

4. **画面サイズ**: レスポンシブテストでは、ブラウザの開発者ツールを使用して画面サイズを変更します。実機でのテストも推奨します。

5. **外部リンク**: ドコモオンラインショップへのリンクは実際の外部サイトに遷移します。テスト環境では`target="_blank"`属性により新しいタブで開きます。

### 既知の制限事項
1. **製品画像**: 現在、製品画像は絵文字（📱）で代替表示されています。実際の製品画像は今後実装予定です。

2. **検索機能**: ヘッダーの検索アイコンは表示されていますが、機能は未実装です（PBI-CM-003で実装予定）。

3. **Android/ドコモ認定リユース品**: これらのカテゴリページは「準備中」メッセージが表示されます。製品データは今後追加予定です。

4. **認証機能**: 現在の認証機能はモック実装です。LocalStorageを使用した簡易的な実装となっています。

5. **APIエンドポイント**: バックエンドAPIのエンドポイントは環境変数で設定可能ですが、実際のAPIが稼働していない場合はエラーが表示されます。
